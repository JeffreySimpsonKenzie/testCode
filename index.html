    Create multiple tags at once.

    In VS Code there are shortcuts to creating multiple tags at once.

    • Example 1:
      li*3
      This will create three list item tags

    • Example 2:
      ul>li*3
      This will create an unordered list tag with 3 list item tags inside.

    • Example 3:
      p.classname#idname
      This will create a paragraph tag with a class and id attributes.
      The . represents the class attribute and # represents the id attribute.

    Try out some of these in VS Code! Try using different types of HTML tags.
